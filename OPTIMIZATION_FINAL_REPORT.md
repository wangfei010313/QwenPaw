```

│  └─ 加载自定义提供商配置
├─ 完成 LocalModelManager 初始化
│  └─ 加载本地模型配置
└─ ✓ 所有功能准备就绪
```

## 🔍 常见问题解答

**Q: 为什么用户一开始看到的是部分功能？**
A: 不是的。Builtin 提供商（OpenAI, DeepSeek, Ollama 等）立即可用。自定义提供商在后台加载，但由于大多数用户使用 builtin 提供商，这个延迟不可见。

**Q: 如果用户在启动过程中访问自定义提供商会怎样？**
A: 系统会自动完成初始化（可能需要等待几秒），然后返回数据。完全安全。

**Q: 这个优化对线上稳定性有影响吗？**
A: 没有。我们只是改变了加载时序，没有改变加载内容。所有功能最终都会初始化。

**Q: 如果我想回到原来的同步初始化怎么办？**
A: 改一行代码即可：`ProviderManager(lazy_init=False)`
